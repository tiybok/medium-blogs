Step to Install Docker-ce on Redhat 8 and run it

Step 1 : Download and Dokcer-ce on RHEL-8(Redhat Enterprise Linux)

Dokcer-ce

Docker is a software platform for building applications based on containers , small and lightweight execution environments that make shared use of the operating system kernel but otherwise run in isolation from one another. While containers as a concept have been around for some time, Docker, an open source project launched in 2013, helped popularize the technology, and has helped drive the trend towards containerization and microservices in software development that has come to be known as cloud-native development.

A] Add the URL of the repository for downloading

Firstly, we need to put the url, from where we have to download docker image. It should be inside the yum repo.

Use command “cd /etc/yum.repos.d/” to go inside the folder.

vim /etc/yum.repos.d/docker.repo

This command “cat <your-repo-name>.repo” will read the file.

cat /etc/yum.repos.d/docker.repo

Add this url in the given format to your repository.

B] Use command to install Docker-ce :

yum install docker-ce --nobest -y

Normally, when installing a package, the best available candidate is selected from a repository. In this case, for example, the installation of the latest version of docker-ce is attempted (and fails).

By using the --nobest option, we can change this behavior so that the first version of docker-ce with satisfiable dependencies is selected as “fallback”.

Put -y when it asks for permission while installing.

C] Disable Firewalls

Step 2 : Start docker service and pull image from the repository and execute it.

A] Use commands :

systemctl enable docker
systemctl status docker

D] Use command:

docker pull centos:8

Use this command to go inside the container

docker run -i -t --name arth_task centos:8

We can see that before we were in baseOS (Redhat 8) and now we go in docker container.

Steps for configuring Apache HTTP WebServer on the top of Docker Container

Step 3 : Install apache webserver and create a Web-Page

Use yum install vim for install Vim in container.

Put -y when it asks for permission while installing.

Vim is a highly configurable text editor built to make creating and changing any kind of text very efficient. It is included as “vi” with most UNIX systems.

Vim is rock stable and is continuously being developed to become even better. Among its features are:

  • persistent, multi-level undo tree
  • extensive plugin system
  • support for hundreds of programming languages and file formats
  • powerful search and replace
  • integrates with many tools

Use yum install httpd for installing apache web-server in container.

yum install http

After successful installation, Now create a webpage. Create a webpage either using gedit or vi or vim editor. Here I am using vim.

We need to create your page inside /var/www/html/ folder. It is a by-default folder , we can change it too. I am using this default folder only. The data written in this file “index.html” will be displayed on the webpage.

Data written inside in index.html :

<b1>This is the index page ! Arth-Task 7.2 </b1>

Step 4 : Start the httpd service and check the webpage

We use command for starting the service is

/usr/sbin/httpd

For checking we can use curl command , so we need to know the ip address of this OS container. We can’t use ifconfig command directly, for this we need to install net-tool using command

yum install net-tools

Net-tool is a tool to diagnose the network and services status. It provides a graphical user interface for a number of common network tools, such as ping, netstat, traceroute, port scan, lookup, finger, and whois.

The functionality is very basic, it is oriented to get the first insight of an potential issue and to bring this information to technical staff.

Now check the ip-address using command

ifconfig

You can curl from the same OS or either use base OS for checking the webpage. Use command curl <ip-address>

curl 172.17.0.2

Successfully Executed! You can we the actual page from web browser using url “172.17.0.2/index.html”

Steps for running Python on the top of Docker Container

Step 5 : Create a new directory named “python3” and go in it

cd python3/

Step 6 : Installation of Python on Container OS

Python is an interpreted, high-level and general-purpose programming language. Python’s design philosophy emphasizes code readability with its notable use of significant indentation.

yum install python3

Check whether it is installed by following command

python3 --version

Go into “python3” directory and create “arth_task.py” python3 file and put code in it.

I have create a small python program to generate a random number between 0 to 999 and to print a random number when runned.

Step 7 : Run Python file on Container OS

Run command to run python3 file and See outputs

Multiple Outputs

python3 arth_task.py

Conclusion

We will have a fairly good understanding of the various components that Dokcer-ce on RHEL-8 (Redhat Enterprise Linux). We successfully configured Apache HTTP WebServer and Python on the top of Docker Container.
